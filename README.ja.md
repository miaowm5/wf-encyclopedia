## ディレクトリ

- `src/`：フロントエンドのソースディレクトリ（Svelte コンポーネント、状態、データベース、UI）。
- `public/`：Vite の公開静的アセット。
- `tool/`：リソース処理スクリプト（ストーリー、ボイス、スプライトシートなど）。
- `app/`：Neutralino デスクトップ版プロジェクトとパッケージ設定。
