## 目录说明

- `src/`：前端源码目录（Svelte 组件、状态、数据库、UI）。
- `public/`：Vite 公共静态资源目录。
- `tool/`：资源处理脚本（剧情、语音、图集等）。
- `app/`：Neutralino 桌面端工程与打包配置。
